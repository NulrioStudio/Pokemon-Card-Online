# Pokemon-Card Online Simulator

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-blue)](https://pokemon-card-online.nulrio.com)

ブラウザだけで動作する、ポケモンカードゲームの非公式オンライン対戦シミュレーターです。
PeerJSを利用したP2P通信により、サーバーを介さずに対戦相手と直接データを同期します。

---

## 🌟 本プロジェクトの目的
このツールは、以下の目的のために制作された非公式のファンプロジェクトです。
- **カードの魅力を広める**: 離れた友人と手軽にデッキの動きを確認し、ポケモンカードの楽しさを共有する。
- **プレイングの練習**: お気に入りのデッキを回す練習を行い、実際の対戦イベントや公式サービスの利用につなげる。
- **技術の探求**: WebRTC (P2P) を用いたリアルタイム同期技術の学習。

## 🌐 デモサイト (GitHub Pages)
**[👉 ここをクリックして対戦を開始する](https://pokemon-card-online.nulrio.com)**

## 🚀 特徴
- **インストール不要**: ブラウザだけで今すぐに対戦を始められます。
- **デッキ自動読み込み**: 公式サイトのデッキコードを入力するだけで、自分のデッキを自動構成します。
- **P2P対戦**: サーバーを介さない直接通信により、低遅延な同期を実現。
- **フルシミュレーション**: ダメカン、状態異常マーカー、コイントス、チャット機能を搭載。

## 🛠 使い方
1. **準備**: [ポケモンカードゲーム公式ホームページ](https://www.pokemon-card.com/)でデッキを作成し、「デッキコード」を取得します。
2. **接続**: デモサイトを開き、自分のデッキIDを入力。発行された自分のIDを相手に教えるか、相手のIDを入力して「対戦開始」を押します。
3. **操作**:
   - **ドラッグ**: カードやダメカンの移動
   - **ダブルクリック**: カードの拡大表示 / カウンターの削除
   - **右クリック**: カードの裏表を反転
   - **VIEWボタン**: 山札やトラッシュの中身を確認

## 📦 技術スタック
- **Frontend**: HTML5 / CSS3 / JavaScript (jQuery 3.7.1)
- **Networking**: [PeerJS](https://peerjs.com/) (P2P/WebRTC)
- **Proxy**: [AllOrigins](https://allorigins.win/) (CORS回避用)
